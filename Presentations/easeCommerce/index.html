    What's going on here?

    Every time you save your experiment, we send an updated JavaScript file that contains your changes to our content delivery network (CDN). Our CDN makes sure that the JavaScript file is served to visitors of your site as fast as possible. The tradeoff when using a CDN however is that it can take some time for the CDN to process updated content.

    If you don't want to wait for our CDN, you can visit a copy of your page that we host. This should pretty closely match the experience of working against your live site.
